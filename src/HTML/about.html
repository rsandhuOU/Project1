This Website was made by Ronveer Sandhu for CSI 3150 Project 1 The Data is used from NBAAPI from Rapid API The Homepage shows the NBA champions from 2016 to present day The Search Player page allows users to search for a string of words which are in a NBA player's last name from 2016 to present day. It will display some available data such as position, height, and experience. The Standings page allows users to view the standings of NBA teams in recent years, and how far back they were from 1st place in their conference The Games page displays yesterday's results, a link to todays games, and what games are tomorrow This website was constructed using html, tailwind CSS, vanillaJS, and GitHub for hosting and version control
